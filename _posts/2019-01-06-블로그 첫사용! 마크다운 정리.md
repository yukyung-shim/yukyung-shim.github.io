## 블로그 첫 포스팅! 마크다운 정리

# 1. 제목 태그(h 사용)
h1 tag : #
h2 tag : ##
h3 tag : ###
...
h7 tab : #######

# 2. 블럭인용문자 : >
> 블럭인용문자 '>' 사용

# 3. 순서있는 목록 : 숫자 + . 이용
1. 순서1
2. 순서2
3. 숫자3
5. 숫자5

'1. 순서1, 2. 순서2, 3. 숫자3, 5. 숫자5'를 입력했지만 입력숫자에 상관없이 내림차순으로 표시된다!



# 4. 순서 없는 목록 : *, +, -사용
* 순서*
  + 순서+
    - 순서-

# 5. 코드사용 : <pre><code></code></pre>
<pre>
  <code>
    <p> 코드부분! </p>
  </code>
</pre>

# 6. 수평선 사용
'* * *', ''***','*****', '- - -', '--------------------------------------- '를 이용하면 아래와같이 수평선이 표시된다.

* * *
***
*****
- - -
---------------------------------------
